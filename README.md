# КалоФото — инструкция по деплою на Railway

## Шаги:

1. Загрузи эти файлы на GitHub (новый репозиторий)
2. Зайди на railway.app
3. New Project → Deploy from GitHub repo → выбери репозиторий
4. В Settings → Variables добавь:
   - ANTHROPIC_API_KEY = (твой ключ с console.anthropic.com)
5. Скопируй URL сайта из Railway (типа https://kalofoto-xxx.railway.app)
6. В Telegram найди @BotFather → /newapp → вставь URL
